Now even easier to use Supervisor

Since I last wrote about supervisor it’s become easier to use thanks to collective.recipe.supervisor. Now it’s as simple as editing your buildout.cfg.

Add a section for Supervisor:

[buildout]
    parts =
        ...
        supervisor

Create the section:

[supervisor]
recipe = collective.recipe.supervisor
programs =
    10 zeo ${zeo:location}/bin/runzeo ${zeo:location}
    20 instance ${instance:location}/bin/runzope ${instance:location} true
